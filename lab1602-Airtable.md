#### **1.2. Интерфейс Airtable (основные элементы)**

| Элемент | Описание |
|---------|----------|
| **Base** | База данных, содержит несколько таблиц (Grid views) |
| **Table** | Таблица, аналогична листу в Excel или таблице SQL |
| **Field** | Колонка с определённым типом данных (текст, число, дата, ссылка) |
| **Record** | Строка в таблице (запись) |
| **View** | Представление (фильтрация, сортировка, группировка) |
| **Form** | Форма для ввода данных, создаётся автоматически |
| **Extension** | Расширения (графики, календари, диаграммы) |
