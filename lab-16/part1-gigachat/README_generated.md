# 📚 Прочитаем вместе: README на русском для проекта на Python

Этот проект содержит три основных функции:

- `is_prime(n)` — проверка числа на простоту.
- `fibonacci(n)` — вычисление n-го числа Фибоначчи.
- `normalize_phone(phone_number)` — нормализация телефонного номера в стандартизированный формат.

## Описание функций и их использование

### 1. Проверка числа на простоту (`is_prime`)
Функция принимает целое число и возвращает логическое значение (True или False), указывающее, является ли это число простым.

**Пример использования:**
```python
print(is_prime(7))   # True
print(is_prime(8))   # False
```

### 2. Вычисление n-го числа Фибоначчи (`fibonacci`)
Функция принимает целое число и возвращает n-е число последовательности Фибоначчи.

**Пример использования:**
```python
print(fibonacci(5))  # 5
print(fibonacci(10)) # 55
```

### 3. Нормализация телефонных номеров (`normalize_phone`)
Функция принимает строку с произвольным телефонным номером и преобразует его к стандартному формату +7 (XXX) XXX-XX-XX.

**Пример использования:**
```python
print(normalize_phone("89161234567"))  # '+7 (916) 123-45-67'
print(normalize_phone("+79161234567")) # '+7 (916) 123-45-67'
```
